Desktop AI Assistant

Get help on any screen, only when you ask.

OnScreen AI Assistant stays available as a floating icon. It captures your screen only when you click, then helps with debugging, docs, and workflows.

Download Installer (.exe) Download Portable

If links fail, build first with npm run dist.

How it works

  1. Launch app and keep floating AI icon on screen.
  2. Click icon to manually select and capture your current screen.
  3. Ask a question and analyze with OpenAI or Gemini.
